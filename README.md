### Install On TERMUX
```bash
> apt update && apt upgrade -y
> pkg install git -y
> pkg install nodejs -y
> git clone https://github.com/MrG3P5/Brainly-Scraper.git
> cd Brainly-Scraper
> npm install
> npm start
```

- Download NodeJS latest in <a href="https://nodejs.org/en/download/current"><font>Here</a>
